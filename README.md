# PolSAR-generation
论文《Azi-Vec: High-resolution Azimuth-Sensitive PolSAR Image Generation for Vehicle Automatic Target Recognition》的代码与相关数据。
相关数据获取方式，其中 :
HH 通道数据见[下载页面](https://www.scidb.cn/file?fid=1d2a817c2267d47a9a5eb74ce86cc79d&mode=front)，
单极化数据集面向高分辨率 SAR 目标识别应用需求，通过场景布设、航线规划和雷达数据采集等流程获取了 4 个场景下的高分辨率 SAR 目标数据，涵盖 7 大类 21 型车辆目标，背景包括水泥地面、草地和树丛，频段覆盖 Ku 和 X 波段，观测视角覆盖 5 个俯仰角和 72 个方位角，共形成 3308 张 SAR 目标切片样本。数据集同时提供 SAR 幅度图像及对应复数据，可为高分辨率 SAR 目标识别、目标散射特性分析及相关方法研究提供数据支撑。
完整数据说明见[说明页面](https://www.sciengine.com/CSD/doi/10.11922/11-6035.csd.2025.0094.zh)。
[全极化数据页面](https://www.scidb.cn/detail?dataSetId=0378a4e05fdf43c4b81187e4448e369c&version=V1) 
全极化数据集覆盖 7 大类 21 型车辆，包含水泥坪、草地和树丛 3 类典型背景，涵盖 Ku 和 X 两个波段、5 个俯仰角以及最多 72 个方位角，共提供 3308 幅 Pauli-RGB 图像及对应的极化散射矩阵数据。该数据集可为车辆目标极化散射特性分析、细粒度识别、小样本学习和域适应等研究提供高质量数据支撑，并有助于推动极化 SAR 目标识别技术的可靠应用。
